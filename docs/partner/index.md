# EIXAM Connect SDK - Partner Documentation

EIXAM Connect is the integrable safety layer of EIXAM. It allows a partner app to add connected SOS, device flows, telemetry, and backend-linked operational behavior without building the full infrastructure from scratch.

This site is intentionally narrow and partner-facing.
