# EIXAM documentation bundle

This bundle is designed to be copied into the `eixam_connect_sdk` monorepo.

It includes:
- `docs/partner` for partner-facing Markdown source
- `docs/full` for full/internal Markdown source
- `site/partner` for generated partner HTML
- `site/full` for generated full/internal HTML
- `config/*.json` for navigation

Rule of work:
- edit Markdown
- regenerate HTML
- do not hand-edit generated HTML
